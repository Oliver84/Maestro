# Audio Files Directory

Place your audio files here for use in cues.

## Supported Formats
- MP3 (recommended for compatibility)
- WAV (high quality, larger file size)
- OGG
- M4A

## Usage

Files placed in this directory can be referenced in cues using:
```
/audio/your-file-name.mp3
```

## Example Structure
```
public/audio/
├── opening-music.mp3
├── scene1-background.mp3
├── narrator-intro.wav
└── closing-music.mp3
```

## For Testing

You can download free sample audio from:
- https://www.soundhelix.com/examples/mp3/
- https://freesound.org/
- Or use any MP3 files you have

Just copy them to this folder and reference them in your cues!
